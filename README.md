# 创始人手册 · 随身读本

《The Founder's Playbook: Building an AI-Native Startup》（Anthropic 出品）中文全本网页阅读器。单文件、自包含，手机浏览器可直接打开。

## 功能

- 中文全本正文（七章 + 资源附录，约2.4万字）
- 按"练习"（可执行练习）与"注/说明"两类知识框还原原书结构
- 章节目录 + 阅读进度自动保存（浏览器本地存储，按设备/浏览器独立记录，不跨设备同步）
- 字号调节、深色/浅色主题切换
- 「与这本书对话」标签页：填入自己的 DeepSeek API Key（存在本地浏览器，不上传任何服务器），基于全书正文流式问答，支持 Markdown 渲染
- 移动端优先，支持左右滑动切章

## 说明

本书文字版权归 Anthropic 所有，本仓库内容整理自公开流通的中文翻译版 PDF，仅用于个人学习与阅读体验研究，不用于商业用途。如权利方认为不当，请联系仓库所有者，会立即处理。

## 技术

单个 `index.html`，无构建步骤、无外部依赖（字体均以 base64 内嵌），可直接用任意静态托管服务部署。由 `book-reader-site` 通用模板（同一套代码也用于《小岛经济学》随身读本）生成。
